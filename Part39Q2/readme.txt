opendir() / readdir(): Це класичний інтерфейс. DIR * — це потік (схожий на FILE *), а struct dirent містить лише мінімум інформації (назву файлу та inode).

lstat() vs stat(): У коді я використав lstat.

stat(): якщо файл є символічним посиланням (symlink), функція покаже інформацію про файл, на який воно вказує.

lstat(): покаже інформацію про саме посилання. Це стандартна поведінка ls -l.

Бітові маски (&): Рядок mode & S_IRUSR перевіряє, чи встановлено біт читання для власника. Це бінарна операція AND. Якщо результат не нуль — право є.

readdir() повертає тільки ім'я файлу (наприклад, file.txt). Щоб отримати інформацію через stat(), потрібно склеїти шлях до папки та ім'я файлу (наприклад, /etc/file.txt), інакше stat не знайде файл.
: Поле st_mode містить біти прав. Ми використаємо макроси S_IRUSR (read user), S_IWUSR (write user) тощо.

икористовуємо <pwd.h> та <grp.h>, щоб перетворити числовий ID (uid) у текстове ім'я (наприклад, root)
