Якою є кількість посилань (link count) каталогу при зміні кількості записів у ньому?
Якщо тільки створити каталог, то кількість посилань = 2(запис у батьківському каталозі(ім'я папки, спеціальний запис . всередині самого каталогу, який посилається на себе).
Але якщо створити ще один folder то кількість (link count) посилань буде 3
Розрахувати Links можна формулою Links = 2 + N де N - кількість підкаталогів.
